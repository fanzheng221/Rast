# @rust_ast/mcp-server

MCP server for Rast AST analysis.

## Install

```bash
npm install -D @rust_ast/mcp-server
```

## Run

```bash
npx @rust_ast/mcp-server
```

## Claude Desktop Example

```json
{
  "mcpServers": {
    "rast": {
      "command": "npx",
      "args": ["-y", "@rust_ast/mcp-server"]
    }
  }
}
```

# @rust_ast/cli

Command-line interface for Rast pattern matching and code rewriting.

## Install

```bash
npm install -g @rust_ast/cli
```

## Usage

```bash
rast --help
```

Or run without global install:

```bash
npx @rust_ast/cli --help
```
